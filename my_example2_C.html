Below is about Paragraph tag

The tag defines a paragraph. Browsers automatically add a single blank line before and after each p element. Tip: Use CSS to style paragraphs.

The p HTML element represents a paragraph. Paragraphs are usually represented in visual media as blocks of text separated from adjacent blocks by blank lines and/or first-line indentation, but HTML paragraphs can be any structural grouping of related content, such as images or form fields.

Below is about div tag

tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. The div tag is easily styled by using the class or id attribute. Any sort of content can be put inside the div tag!
